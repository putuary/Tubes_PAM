# Deskripsi Aplikasi
Aplikasi yang kami buat memiliki fitur menampilkan daftar dan detail dari trending film, serial tv, dan film. Data tersebut diambil dari API piak ketiga yaitu The Movie Database (https://developers.themoviedb.org/3/getting-started/introduction). Aplikasi ini memanggil api dari TMDB menggunakan redux yang dipanggil secara asinkron. </br>

# Cara menginstall
1. Klik 119140030-119140030-DaftarFilm.apk 
2. Klik install
3. Lalu buka aplikasi yang bernama DaftarFilm
